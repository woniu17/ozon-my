# 卖家信息与评级

> Ozon Seller API — 卖家信息与评级
>
> 基础地址：`https://api-seller.ozon.ru`
>
> 认证请求头：`Client-Id`、`Api-Key`、`Content-Type: application/json`

---

## 目录

**API密钥相关信息**（1）

- `POST` `/v1/roles` — 使用API密钥获取角色和方式列表

**卖家账户信息**（2）

- `POST` `/v1/seller/info` — 卖家个人中心信息
- `POST` `/v1/seller/ozon-logistics/info` — Ozon配送开通信息

**卖家评级**（2）

- `POST` `/v1/rating/index/fbs/info` — 获取错误指数：FBS 和 rFBS
- `POST` `/v1/rating/index/fbs/posting/list` — 影响错误指数的货件列表：FBS 和 rFBS

---

## API密钥相关信息

### `POST` `/v1/roles`

**使用API密钥获取角色和方式列表**

`operationId: AccessAPI_RolesByToken`

获取与API密钥绑定的角色和方式信息的方式。

#### 响应（`v1RolesByTokenResponse`）

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| expires_at | string | 否 | 密钥到期日期。 |
| roles | array<object> | 否 | 可用角色和方式信息。 |
|   roles[].name | string | 否 | 角色名称。 |
|   roles[].methods | array<string> | 否 | 角色可用的方式。 |

## 卖家账户信息

### `POST` `/v1/seller/info`

**卖家个人中心信息**

`operationId: SellerAPI_SellerInfo`

#### 响应（`v1SellerInfoResponse`）

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| company | object | 否 | 公司。 |
|   company.country | string | 否 | 国家。 |
|   company.currency | string | 否 | 货币： - `RUB`——俄罗斯卢布； - `EUR`——欧元； - `USD`——美元； - `CNY`——人民币； - `BYN`——白俄罗斯卢布； - `KZT`——坚戈； - `KGS`——吉尔吉斯斯坦索姆。 |
|   company.inn | string | 否 | 税号（INN）。 |
|   company.legal_name | string | 否 | 法人名称。 |
|   company.name | string | 否 | Ozon上的公司名称。 |
|   company.ogrn | string | 否 | 国家基本登记号（OGRN）。 |
|   company.ownership_form | string | 否 | 所有制形式。 |
|   company.tax_system | string | 否 | 税收制度： - `UNKNOWN`——未知； - `UNSPECIFIED`——未指定； - `OSNO`——普通税制（OSNO）； - `USN`——简化税制（USN）； - `NPD`——职业收入税（NPD）； - `AUSN`——自动简化税制（AUSN）； - `PSN`——专利税制（PSN）。 |
| ratings | array<object> | 否 | 评级列表。 |
|   ratings[].current_value | object | 否 | 评级值。 |
|     ratings[].current_value.date_from | string | 否 | 评级统计开始日期。 |
|     ratings[].current_value.date_to | string | 否 | 评级统计结束日期。 |
|     ratings[].current_value.formatted | string | 否 | 格式化后的评级值。 |
|     ratings[].current_value.status | object | 否 | 评级状态。 |
|       ratings[].current_value.status.danger | boolean | 否 | 指示是否已超过封禁评级阈值。 |
|       ratings[].current_value.status.premium | boolean | 否 | 指示是否已达到参与高级计划的阈值。 |
|       ratings[].current_value.status.warning | boolean | 否 | 指示是否已发出关于可能超过封禁阈值的警告。 |
|     ratings[].current_value.value | number | 否 | 系统中的评级值。 |
|   ratings[].name | string | 否 | 评级名称。 |
|   ratings[].past_value | object | 否 | 上一次的评级值。 |
|     ratings[].past_value.date_from | string | 否 | 评级统计开始日期。 |
|     ratings[].past_value.date_to | string | 否 | 评级统计结束日期。 |
|     ratings[].past_value.formatted | string | 否 | 格式化后的评级值。 |
|     ratings[].past_value.status | object | 否 | 评级状态。 |
|       ratings[].past_value.status.danger | boolean | 否 | 指示是否已超过封禁评级阈值。 |
|       ratings[].past_value.status.premium | boolean | 否 | 指示是否已达到参与高级计划的阈值。 |
|       ratings[].past_value.status.warning | boolean | 否 | 指示是否已发出关于可能超过封禁阈值的警告。 |
|     ratings[].past_value.value | number | 否 | 系统中的评级值。 |
|   ratings[].rating | string | 否 | 系统中的评级名称。 |
|   ratings[].status | string | 否 | 评级状态： - `UNKNOWN`——未指定； - `OK`——良好； - `WARNING`——指标需关注； - `CRITICAL`——严重。 |
|   ratings[].value_type | string | 否 | 值的类型： - `UNKNOWN`——未指定；  - `INDEX`——指数； - `PERCENT`——百分比； - `TIME`——时间； - `RATIO`——系数； - `REVIEW_SCORE`——评分； - `COUNT`——数量。 |
| subscription | object | 否 | 订阅。 |
|   subscription.is_premium | boolean | 否 | `true`，前提是有订阅。 |
|   subscription.type | string | 否 | 订阅类型： - `UNKNOWN`——未指定； - `UNSPECIFIED`——无需订阅； - `PREMIUM`——Premium； - `PREMIUM_LITE`——Premium Lite； - `PREMIUM_PLUS`——Premium Plus； - `PREMIUM_PRO`——Premium Pro。 |

### `POST` `/v1/seller/ozon-logistics/info`

**Ozon配送开通信息**

`operationId: SellerAPI_SellerOzonLogisticsInfo`

#### 响应（`v1SellerOzonLogisticsInfoResponse`）

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| available_schemas | array<string> | 否 | 可用模式类型： - `UNKNOWN`——未指定； - `FBO`——从Ozon仓库配送； - `FBS`——从自有仓库配送。 |
| ozon_logistics_enabled | boolean | 否 | `true`，表示Ozon配送已开通。 |

## 卖家评级

### `POST` `/v1/rating/index/fbs/info`

**获取错误指数：FBS 和 rFBS**

`operationId: RatingAPI_GetFBSRatingIndexInfoV1`

#### 响应（`v1GetFBSRatingIndexInfoV1Response`）

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| currency_code | string | 否 | 错误处理费用的币种代码。 |
| defects | array<object> | 否 | 按天计算的错误指数。 |
|   defects[].date | string | 否 | 日期格式为`YYYY-MM-DD`。 |
|   defects[].index_by_date | number | 否 | 错误指数的数值。 |
|   defects[].processing_costs_sum_by_date | number | 否 | 错误处理费用。 |
| index | number | 否 | 周期内的错误指数数值。 |
| period_from | string | 否 | 计算周期开始日期（格式`YYYY-MM-DD`）。 |
| period_to | string | 否 | 计算周期结束日期（格式`YYYY-MM-DD`）。 |
| processing_costs_sum | number | 否 | 周期内的错误处理费用。 |

### `POST` `/v1/rating/index/fbs/posting/list`

**影响错误指数的货件列表：FBS 和 rFBS**

`operationId: RatingAPI_ListFBSRatingIndexPostingsV1`

#### 请求体（`v1ListFBSRatingIndexPostingsV1Request`）

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| cursor | string | 否 | 用于获取下一批数据的指针。 |
| filter | object | 是 | 筛选器。 |
|   filter.date_from | string | 是 | 周期开始日期。 |
|   filter.date_to | string | 是 | 周期结束日期。 |
|   filter.posting_numbers | array<string> | 否 | 货件编号。 |
| limit | integer | 是 | 返回结果中的数值数量。 |

#### 响应（`v1ListFBSRatingIndexPostingsV1Response`）

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| cursor | string | 否 | 用于获取下一批数据的指针。 |
| errors | array<object> | 否 | 影响错误指数的货件。 |
|   errors[].charge_percent | number | 否 | 处理费用占货件价值的百分比。 |
|   errors[].charge_price | number | 否 | 错误处理费用金额。 |
|   errors[].charge_price_currency_code | string | 否 | 错误处理费用的币种代码： - `RUB`——俄罗斯卢布， - `BYN`——白俄罗斯卢布， - `KZT`——坚戈， - `EUR`——欧元， - `USD`——美元， - `CNY`——人民币。 |
|   errors[].delivery_schema | string | 否 | 配送方案： - `FBS`,  - `rFBS`,  - `erFBS`. |
|   errors[].error_at | string | 否 | 出现错误的日期。 |
|   errors[].has_grace_status | boolean | 否 | `true`，表示货件享有优惠状态。 |
|   errors[].index | number | 否 | 错误指数的数值。 |
|   errors[].posting_error_type | string | 否 | 错误类型： - `UNSPECIFIED`——未指定； - `SELLER_CANCELLATION`——卖家取消； - `SELLER_DELAY`——卖家逾期。 |
|   errors[].posting_number | string | 否 | 货件编号。 |
|   errors[].product_price | number | 否 | 货件中的商品价值。 |
|   errors[].product_price_currency_code | string | 否 | 商品价值的币种代码： - `RUB`——俄罗斯卢布， - `BYN`——白俄罗斯卢布， - `KZT`——坚戈， - `EUR`——欧元， - `USD`——美元， - `CNY`——人民币。 |
| has_next | boolean | 否 | `true`，表示查询结果未包含所有货件。 |
