### `POST` `/v1/product/import-by-sku`

**通过SKU创建商品**

`operationId: ProductAPI_ImportProductsBySKU`

该方法会创建指定SKU的商品卡片副本。  该方法仅支持复制其他卖家的商品卡片，无法复制您自己的商品。  如果卖家[禁止复制](https://global-help.ozon.com/zh/products/upload/upload-types/copying/?region=CHN)， 将无法创建卡片副本。  无法通过SKU更新商品。  该方法对每分钟和每天的商品操作数量有限制。如果超过限制，将返回`429`错误，错误说明位于`message`字段中，同时响应会包含以下响应头： - `Item-Retry-After`——距离限制更新还剩多少分钟。对于每日限制，表示距离莫斯科时间03:00还剩多少时间。 - `Item-Rate-Limit-Remaining`——距离下一次限制重置前还可执行的操作数量。

|   items[].sku | integer | 是 | 在Ozon系统中的商品ID — SKU。 |

**品列表的**

`operationId: ProductAPI_GetProductList`

用于获取所有商品列表的方法。  如果使用 `offer_id` 或 `product_id` 进行筛选，则无需填写其他参数。 将返回具有指定标识符的所有商品，无论其是否可见。 每次请求只能使用一组标识符，且商品数量不能超过 1000 个。  如果不使用标识符进行查询，则需在后续请求中指定 `limit` 和 `last_id`。
