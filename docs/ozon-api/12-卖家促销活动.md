### `POST` `/v1/seller-actions/create/voucher`

**创建采用"促销码折扣"机制的促销活动**

`operationId: SellerActionsCreateVoucher`

您可以在Ozon for dev开发者社区的[评论区](https://dev.ozon.ru/community/1872-Novye-metody-dlia-raboty-s-aktsiiami-sellera)对该方法的使用情况留下反馈。

#### 请求体（`v1SellerActionsCreateVoucherRequest`）

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| budget | integer | 是 | 促销活动预算。预算用尽后，促销活动将停止。 |
| date_end | string | 是 | 促销活动结束日期与时间。 |
| date_start | string | 是 | 促销活动开始日期与时间。 |
| discount_type | string | 是 | 折扣类型：  - `PERCENT`——百分比折扣；  - `CURRENCY`——按金额折扣。 |
| discount_value | number | 是 | 折扣幅度。 |
| title | string | 是 | 促销活动名称。 |
| user_ids | array<string> | 否 | 可使用该促销码的用户标识符列表。 |
| voucher_parameters | object | 是 | 促销码参数。 |
|   voucher_parameters.count_codes | integer | 是 | 促销码数量。 |
|   voucher_parameters.is_private | boolean | 是 | `true`，表示促销码公开可用。 |
|   voucher_parameters.type | string | 是 | 促销码类型：  - `ONE`——面向所有买家的促销码，可用于1个订单；  - `MULTIPLE`——面向所有买家的促销码，可用于任意数量的订单；  - `UNIQUE`——面向1位买家的促销码，可用于1个订单。 |

#### 响应（`v1SellerActionsCreateVoucherResponse`）

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| action_id | integer | 否 | 促销活动标识符。 |
