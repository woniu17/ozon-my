批量上架 · 使用说明
1
登录 Ozon 卖家中心

在浏览器里打开 seller.ozon.ru 任意页面并保持登录，别关。我们用你的登录态抓数据。

2
粘贴 SKU + 价格

每行一条，最简只要 SKU + 售价 两列：

1234567890,99.9
1234567890,99.9,JZ-001
1234567890,99.9,150,200,180,80

SKU 在哪找？ Ozon 商品页 URL ozon.ru/product/123456789 末尾那串数字就是。 其他字段（名/图/重量/类目）会自动从 Ozon 公开页抓。

3
选店铺 → 一键提交
  • 「目标店铺」点击 pill 多选店铺，支持搜索 / 国家分组
  • 品牌 / 图片顺序 / 货币用默认值即可
  • 想加水印、AI 大模型改图、SEO 改写 → 展开「AI 增强」
  • 点「开始批采 + 上架」，结果在下方进度面板逐条显示
数据源说明
  • 主源：ozon.ru 公开商品页 → 商品名、图片、面包屑
  • 辅源：seller.ozon.ru 卖家中心 → 找你卖过的同款，复用属性

任一源命中就能上架。新 SKU 完全没卖过也能正常工作。

常见问题
「未登录 MY」
关掉本窗口，去扩展弹窗（点扩展图标）登录后再打开本页。
「未检测到 seller.ozon.ru tab」
先打开一个 seller.ozon.ru 标签页并登录，再回本页提交。
「所有公开端点都失败」/「页面解析失败」
Ozon 反爬触发，或 SKU 已下架。换个 SKU 或稍后再试。
「未在 search-variant-model 找到匹配」
仅辅源没命中，主源拿到数据就能继续。如果两个都没拿到，看上一条。
提交后想看进度？
提交完成后点顶栏「历史记录」，或前往MY ERP「商品 → 上架记录」。
想批量上库存？
本页默认 0 库存，请单独走MY ERP「商品列表 → 库存导入」。
单批最多多少条？
Ozon 后端限制单批 ≤200 条，超过请分批。
